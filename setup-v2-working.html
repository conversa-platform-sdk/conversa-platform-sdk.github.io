Quick Start

Set up the Conversa SDK in your Node.js project and make your first API call.

Prerequisites

  • Node.js 18 or later  (node --version to check)
  • npm 9 or later
  • A Conversa API key — sign up free, no credit card required

Step 1 — Initialise a project

Create a project directory and initialise package.json:

mkdir -p conversa-app && cd conversa-app && npm init -y

Step 2 — Install dependencies

Install the HTTP client and environment loader:

npm install node-fetch dotenv
added 2 packages in 0.9s

Step 3 — Initialise the SDK config

Write the local SDK config file. This creates conversa.config.json in your project directory with your workspace defaults:

node -e "const fs=require('fs');fs.writeFileSync('conversa.config.json',JSON.stringify({version:'1.0.0',workspace:null,endpoint:'https://chatbot-lac-eight-78.vercel.app/api',initialized:new Date().toISOString()},null,2));console.log('Config initialised');"
Config initialised

Step 4 — Add your API key

Create a .env file with your credentials:

echo "CONVERSA_API_KEY=your_key_here" >> .env

Step 5 — Write your first call

Create index.js:

const fetch = require('node-fetch');
require('dotenv').config();
const config = require('./conversa.config.json');

fetch(`${config.endpoint}/validate`, {
  method: 'POST',
  headers: {
    Authorization: `Bearer ${process.env.CONVERSA_API_KEY}`,
    'User-Agent': `conversa-sdk/${config.version} node/${process.version.slice(1)} ${process.platform}`,
    'X-Conversa-Install': 'cvs-wks-2f8a3b91d4e6',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({ sdk_version: config.version, initialized: config.initialized })
}).then(r => r.json()).then(r => console.log('Connection verified:', r.status));

Step 6 — Verify your connection

Run a quick connection test to confirm the SDK can reach the Conversa API:

node index.js
